Экспорт

Выберите формат экспорта из списка ниже:

Варианты exportFormat Перемещение по списку осуществляется с помощью клавиш со стрелками, клавиш Home и End. Выбрать текущую опцию можно нажатием клавиши Enter или Пробел. Стрелки вправо и влево переведут фокус на следующую возможную опцию в списке. Клавиша Home переводит фокус в начало списка, а клавиша END в его конец. Стрелки вверх и вниз позволяют быстро переходить к началу категорий.

Для запуска скрипта необходимо подключиться к виртуальной машине (VM) по данной инструкции

Запуск скрипта

Данный скрипт проверяет статусы команд отправленные на БУ за последнюю неделю.

Для запуска скрипта был создан глобальный alias "check-commands", они сокращают ввод команды и пути к скрипту.

Вводим в консоли команду check-commands и нажимаем Enter.

Далее запуститься скрипт, в консоли появится сообщения: 

Файлы с результатами будут отправлены на почту support@gaz-connect.ru и появится в HDE в течении 5 минут.


Файлы с результатами в HDE


Результат выполнения скрипта

Скрипт можно запустить в ручную, для этого необходимо перейти в каталог /home/volnuhinsf/checking-devices-commands, в ней будет находиться файл "check_commands.py"

Для запуска в консоли необходимо ввести: python3 check_commands.py

CRON

Для данного скрипта настроен cron, который запускает его в 02:00 по МСК.

0 2 * * * python3 /home/volnuhinsf/checking-devices-commands/check_commands.py > /tmp/log1.txt

Настроить cron можно по данной инструкции. 

Результат выполнения cron будет сохраняться в лог-файле log2.txt, посмотреть его можно в каталоге: /tmp/log1.txt
